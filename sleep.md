<수면진단기능>
1.총 수면 시간(Total Sleep Time, TST)
● 정상 기준
성인 기준: 7–9시간
● 위험 신호
6시간 이하: 수면 부족(chronically insufficient sleep)
10시간 이상: 과수면(Hypersomnia), 우울/수면 장애 의심
2. 수면 효율(Sleep Efficiency, SE)
SE = (실제 잠든 시간 / 침대에 누워있던 시간) × 100
● 정상
≥ 85%
● 경계(의심)
75–84%
● 위험
< 75%
→ 불면증(insomnia), 수면 중 각성 빈번
3. 수면 단계 비율(Sleep Architecture)
정상적인 수면은 N1 → N2 → N3(Deep Sleep) → REM의 반복 구조.
● 위험 기준
N3 < 10%: 깊은 수면 부족 → 피로, 집중력 저하
REM < 15%: 스트레스, 우울, 수면장애 가능성
REM 과도 증가(>35%): 악몽·불안·우울 의심
4.수면 중 생리신호 안정성 (Wearable/센서 기반 진단 시 중요)
● 심박수(HR)
정상: 수면 중 40–70 bpm
위험:
> 85 bpm 지속 → 스트레스, 무호흡 의심
< 40 bpm → 서맥 가능
● 호흡수(RR)
정상: 12–20회/min
위험:
10 이하, 25 이상
불규칙 호흡 → 무호흡/저호흡 가능
● SpO₂
정상: ≥ 95%
위험:
90–94% = 경계
< 90% = 저산소증 → 즉시 경고/신고 기능 필요

< 즉각 경고(High-Risk)>
SpO₂ < 90%
HR < 40 bpm 또는 > 120 bpm
무호흡(Apnea) 지속 > 10초 반복
수면 효율 < 70%
1시간당 각성 15회 이상
< 주의 필요(Mid-Risk)>
깊은 수면(N3) < 10%
총 수면 시간 < 6시간
수면 효율 75–85%
